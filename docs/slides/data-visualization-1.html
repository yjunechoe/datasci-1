2 The basic ggplot

  1. Using your data
  2. define how variables in your dataset are mapped to visual properties (aesthetics)
  3. determine the geometrical object that a plot uses to represent data (geom)

2.1 1 data

Use ratings data

ggplot(
    data = ratings
 )

2.2 2 aesthetic mapping

Map Frequency to x-axis and meanFamiliarity to y-axis.

ggplot(
    data = ratings,
    mapping = aes(
        x = Frequency, 
        y = meanFamiliarity
    )
 )

2.3 3 geom

Represent each value with a point.

ggplot(
    data = ratings,
    mapping = aes(
        x = Frequency, 
        y = meanFamiliarity
    )
 ) +
  geom_point()

3 Adding to the basics

3.1 Mapping categorical variables

When a categorical variable is mapped to an aesthetic, ggplot2 will automatically assign a unique value of the aesthetic (here color) ‚Ä¶ a process known as scaling. ‚Äì R4DS

ggplot(
    data = ratings,
    mapping = aes(
        x = Frequency, 
        y = meanFamiliarity,
        color = Class
    )
 ) +
  geom_point()

3.2 Global vs. local aesthetics

  • globally in ggplot(), which are passed down to all geoms
  • locally in geom_*() which are used by that geom only

. . .

ggplot(
    data = ratings,
    mapping = aes(
        x = Frequency, 
        y = meanFamiliarity
    )
 ) +
  geom_point( 
    mapping = aes(color = Class)
    )

3.3 Mapping vs. setting aesthetics

  • mapping allows us to determine a geom‚Äôs aesthetics based on a variable, and is passed as argument in aes()
  • setting allows us to set a geom‚Äôs aestheics to a constant value (not based on any variable), and passed as argument in geom_*() directly

. . .

ggplot(
    data = ratings,
    mapping = aes(
        x = Frequency, 
        y = meanFamiliarity
    )
 ) +
  geom_point( 
    mapping = aes(color = Class), 
    size = 3
    )

5 Theme and adjusting scaling

5.1 themes

Apply classic theme with base_size 20.

  ) +

5.2 scales: changing color

Remember: When a categorical variable is mapped to an aesthetic, ggplot2 will automatically assign a unique value of the aesthetic (here color) ‚Ä¶ a process known as scaling. ‚Äì R4DS

    mapping = aes(
        x = Frequency, 
        y = meanFamiliarity
    )
 ) +
  geom_point( 
    mapping = aes(color = Class), 
    size = 3
    ) +

7 Geometric objects

7.1 geom_*() aka geoms

There are many. We will start with these, and add a few additional geoms as we move through the course:

geom_histogram() histogram, distribution of a continuous variable
geom_density() distribution of a continuous variable
geom_bar() distribution of a categorical data
geom_point() scatterplot
geom_smooth() smoothed line of best fit

7.2 geom_histogram()

A histogram divides the x-axis into equally spaced bins and then uses the height of a bar to display the number of observations that fall in each bin. ‚Äì R4DS

. . .

ggplot(
    data = ratings,
    mapping = aes(
        x = meanFamiliarity
    )
) + 
    geom_histogram() 
`stat_bin()` using `bins = 30`. Pick better value with `binwidth`.

7.3 geom_histogram()

ggplot(
